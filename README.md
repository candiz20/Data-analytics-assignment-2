# Data-analytics-assignment-2
1. Design a data analytics system to analyze student attendance records and identify attendance shortages.
a) Read dataset using pandas.
b) Calculate attendance percentage for each student.
c) Identify students with attendance < 75%.
d) Perform subject-wise attendance analysis.
e) Visualize attendance percentage using a bar chart.

2. Develop a sales analytics program to analyze product performance and revenue trends.
a) Calculate daily sales revenue.
b) Compute monthly sales.
c) Identify most sold product (based on quantity).
d) Perform category-wise revenue analysis.
e) Visualize monthly sales using a line chart.
